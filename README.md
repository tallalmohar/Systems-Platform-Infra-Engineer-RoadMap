# ML Infrastructure Career Roadmap

An interactive 8-month learning plan to become frontier lab ready in ML infrastructure and training platforms.

## Features

- 5 structured learning phases
- 3 high-signal portfolio projects
- Curated resources from MIT, Stanford, and industry experts
- Interactive timeline and expandable resource cards
- Dark-themed, terminal-inspired UI

## Tech Stack

- React 18
- TypeScript
- Vite
- Pure CSS (inline styles)

## Getting Started

### Install dependencies
```bash
npm install
```

### Run development server
```bash
npm run dev
```

Visit [http://localhost:5173](http://localhost:5173) to view the app.

### Build for production
```bash
npm run build
```

### Preview production build
```bash
npm run preview
```

## Development

The app is a single-page React application with no external dependencies beyond React itself. All styling is done with inline styles for simplicity and portability.
